# Orly Iluz & Co — Real Estate Law (Landing)

One-page, RTL (Hebrew) marketing site built with React + Vite + TypeScript + Tailwind CSS.

## Prerequisites
- Node.js (LTS recommended)

## Setup
```bash
npm install
```

Create a `.env` file (or use `.env.example`):
```bash
cp .env.example .env
```

## Run
```bash
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Lead form
The contact form submits to a Zapier/Make webhook URL via `VITE_LEAD_WEBHOOK_URL`.
